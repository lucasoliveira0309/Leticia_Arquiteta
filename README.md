# 🏡 Landing Page - Arquiteta

Uma Landing Page (One Page) moderna e responsiva desenvolvida para apresentar os serviços de uma arquiteta, facilitando o contato com clientes por meio de diferentes canais.

## ✨ Funcionalidades

- 📱 Design responsivo para celulares, tablets e computadores
- 🎨 Layout moderno e elegante
- 📷 Link para Instagram
- 💬 Botão de contato via WhatsApp
- 📝 Formulário de contato
- ⚡ Navegação rápida em uma única página (One Page)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript

## 📂 Estrutura do Projeto

```
/
├── index.html
├── css/
├── js/
├── img/
└── README.md
```

## 📸 Objetivo

Este projeto foi desenvolvido para oferecer uma presença online profissional para uma arquiteta, reunindo informações, redes sociais e canais de contato em uma única página simples, rápida e intuitiva.
